Project One Details

Overview:
Stroke is a critical health conditon, and early prevention can safe lives. This project uses machine learning algorithms to predict stroke risk using patient data such as age, gender, hypertensions, heart disease, glucose level, BMI, and smoking status.

  • Data Preprocessing and cleaning ( handling missing values,outliers, encoding categorical variables )
  • Exploratory data analysis ( Visualizations, correlations )
  • Feature Scaling and Balancing ( using SMOTE for imbalance classes)
  • Model training and evaulation with Random Forest, Logistic Regression, and Support Vector Machine
  • Hyperparameter tuning, using RandomizedSearchCV for optimized model performance
